## How to Host These Documents

### Option 1: GitHub Pages (Recommended - Free)

1. Create a new repository: `echelon-legal`
2. Add both HTML files: `privacy.html` and `terms.html`
3. Enable GitHub Pages in Settings → Pages
4. Your URLs will be:
   - `https://[username].github.io/echelon-legal/privacy.html`
   - `https://[username].github.io/echelon-legal/terms.html`

### Option 2: Simple Static Host

Create an `index.html` that links to both:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Echelon Legal</title>
</head>
<body style="font-family: -apple-system, sans-serif; padding: 40px; text-align: center;">
    <h1>Echelon Legal Documents</h1>
    <p><a href="/privacy.html">Privacy Policy</a></p>
    <p><a href="/terms.html">Terms of Use</a></p>
</body>
</html>
```

Upload to: Netlify, Vercel, GitHub Pages, or any web host.

## ⚠️ Important Customization Notes

Before publishing these documents, **REPLACE**:

1. **Email addresses:** Update `privacy@echelonapp.com` and `support@echelonapp.com`
2. **Website URLs:** Update `echelonapp.com` with your actual domain
3. **Pricing:** Verify current subscription prices match
4. **Dates:** Update "Last Updated" to current date
5. **Governing Law:** Add your state/country in Section 12
6. **Company Info:** Add your legal business name if applicable

## After Hosting

1. Test both URLs in a browser
2. Update URLs in `lib/screens/paywall_screen.dart`:
   ```dart
   static const String _privacyPolicyUrl = 'https://your-actual-url.com/privacy.html';
   static const String _termsOfUseUrl = 'https://your-actual-url.com/terms.html';
   ```

3. Add URLs to App Store Connect:
   - Privacy Policy URL field
   - EULA/Terms field (or App Description)

4. Test the links work in your app before resubmission!
